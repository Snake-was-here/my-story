# META-LEARNINGS.md

Cross-campaign insights. This file **survives archives** and compounds forever.

After every campaign archive, Ember extracts 3–5 transferable insights and adds them here. Not product-specific. Cross-channel, cross-audience, cross-tactic patterns.

---

## Why This File Exists

Without this, every new campaign starts cold. Ember would re-learn the same lessons on Lively, on Product 2, on Product 3.

With this, Ember gets smarter every campaign.

## Format

Each entry includes:

```
### [YYYY-MM-DD] — [1-line insight]
**From:** [campaign name]
**Evidence:** [specific numbers that made this a real pattern, not a guess]
**How to apply:** [concrete action for future campaigns]
**Confidence:** [low / medium / high]
```

---

## Example Entries (hypothetical, for format reference)

```
### 2026-05-20 — Pinterest outperforms Reddit 3x for visual products under €20
**From:** Lively campaign
**Evidence:** Pinterest: 47 pins → 340 clicks → 18 conversions. Reddit: 30 comments → 95 clicks → 6 conversions. ROI 3.2x on Pinterest.
**How to apply:** For any visual product under €20, start with Pinterest before Reddit.
**Confidence:** medium (one campaign)

### 2026-05-20 — Cold DMs with a specific number in first line get ~3x reply rate
**From:** Lively campaign
**Evidence:** "Saw your comment about old photos — noticed you've got 47 in your profile" type messages got 12% reply vs 4% for generic openers.
**How to apply:** Always include a specific scraped detail in cold DM openers.
**Confidence:** high
```

---

## Confidence Guide

- **Low:** Observed once, could be coincidence
- **Medium:** Observed twice or with moderate sample size (50+ data points)
- **High:** Observed 3+ times or with large sample (200+ data points), consistent direction

Low-confidence entries get re-tested before trusted.

---

## Review Cadence

Ember re-reads this file during every 3-day self-upgrade. Updates confidence levels if new evidence comes in. Removes entries proven wrong.
